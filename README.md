# Vuyo Portfolio

Personal portfolio website for Vuyo Makasana.

This project showcases my work, current tech stack, background, and selected projects in a clean, responsive portfolio built for mobile, tablet, and desktop screens.

## About

I am a self-taught full-stack and mobile developer with a strong frontend foundation and growing backend experience.

Current stack:

- HTML
- CSS
- JavaScript
- React (Intermediate)
- React Native (Intermediate)
- Node.js (Learning)
- Express.js (Learning)

I also use AI tools such as GitHub Copilot and ChatGPT to speed up development while still making sure I fully understand and own the code I ship.

## Pages

- `index.html` - Home page with intro, featured sections, tech stack, projects, and skills
- `about.html` - Personal background, development journey, GitHub section, and skills
- `work.html` - Project showcase page
- `style.css` - Shared styling and responsive layout rules
- `script.js` - Mobile navigation behavior

## Features

- Responsive layout for phones, tablets, laptops, and large screens
- Mobile hamburger navigation
- Skills section with animated progress bars
- Project showcase with live demo and source code links
- Tech stack section covering frontend, mobile, backend, and tools
- Clean dark theme UI

## Running Locally

This is a static website, so no package installation is required.

Options:

- Open `index.html` directly in your browser
- Or run it with a local live server in VS Code for easier development

## Deployment

This project can be deployed easily to platforms such as:
